# Credits & Third-Party Resources — Workout Timer

## Exercise Images

- **Source**: [free-exercise-db](https://github.com/yuhonas/free-exercise-db)
- **License**: [Unlicense](https://unlicense.org/) (public domain)
- **Count**: 873 exercises with JPG images
- **Usage**: Images resized to 320px thumbnails. Only the first image (0.jpg) per exercise is used.
- **Missing images**: Burpees, Russian Twist, Mountain Climbers, High Knees are not available in the source database. Users can add their own images via the "Add Exercise" or "Pick Image" features.

## Sound Effects

All sounds are synthesized in-app and are original works. No third-party sound files are bundled.

| File | Purpose | Description |
|------|---------|-------------|
| `beep.ogg` | Countdown tick / transition | Short sine wave beep (4.5 kHz, 100 ms) |
| `round_start.ogg` | Round / exercise start | Double rising tone (880 Hz + 1100 Hz) |
| `round_end.ogg` | Round / exercise end | Single clear bell tone (700 Hz) |
| `exercise_complete.ogg` | Exercise completed | Rising two-tone chime (C5 + E5) |
| `training_complete.ogg` | Training session finished | Three-tone ascending fanfare (C5 + E5 + G5) |
